# devctx
Repo use devctx. Save token. Caveman style.
## Rule
1. Read `.aictx/CURRENT.md`. Fast state.
2. Write `.aictx/HANDOFF.md`. Tell next agent.
## Map
`.aictx/CURRENT.md`: Now.
`.aictx/HANDOFF.md`: Handoff.
`.aictx/decisions/`: Big choice.
`.aictx/threads/`: Chat.
`.aictx/open/`: Blocked.
